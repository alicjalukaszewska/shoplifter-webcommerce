# Shoplifter
A fake E-Commerce website, created using Angular 13. 

## Technologies used:
- Angular 13
- Angular Material
- Typescript
- SCSS
- HTML5

## Development server
Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
